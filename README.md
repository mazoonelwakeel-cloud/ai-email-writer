# AI Email Writer

This project is an AI Email Writer that helps users create emails quickly using AI.

The user enters the purpose of the email, who it is for, the sender name, language, tone, and length. The program then generates a complete email based on the information provided.

I first created a terminal version using Python, then developed a web version using Flask, HTML, CSS, and JavaScript to make it easier to use.

## Features

* Generate emails with AI
* Choose between different languages
* Select email tone (Formal, Friendly, or Polite)
* Choose email length
* Simple and easy-to-use interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* NVIDIA AI API

## How to Run

1. Install the required libraries:

pip install flask requests python-dotenv

2. Create a `.env` file and add your NVIDIA API key:

API_KEY=your_api_key_here

3. Run:

python app.py

4. Open:

http://127.0.0.1:5000
